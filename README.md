# Проект: Farus

## Обзор

_Две страницы адаптивной верстки, выполненные по заказу компании [Digital-агентство F5](https://f5-studio.ru/),
для внедрения их на сайте компании [Фарус](https://farus.su/)._
Переключение между страницами по клику на главный логотип.
